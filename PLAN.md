# Project Plan — FYP Collaborative AI Chatbot

This is the initial development plan for the FYP collaborative chatbot project, continuing from the previous student's work.

---

## Features in Scope

**Messaging Platform**
- Real-time messaging with notifications
- Tagging feature — users can mention others in messages
- Mobile application

**AI Improvements**
- Group and individual student summary (individual view restricted to coordinator and supervisor)
- Summary history — show current and previous summary side by side for progress tracking
- Student contribution input alongside AI summary
- Reduce AI hallucinations by improving how documents are retrieved and used

---

## Development Phases

**Phase 1 — Setup & Baseline**
Get the existing codebase running locally and verify all current features work before making any changes.

**Phase 2 — AI Improvement**
Improve how the AI retrieves and uses documents when generating responses and summaries. Measure improvement against a baseline recorded in Phase 1.

**Phase 3 — Real-Time & Notifications**
Replace the current messaging approach with real-time communication. Add mention tagging and in-app notifications.

**Phase 4 — Summary Improvements**
Add individual student summaries, summary history comparison, and student contribution input.

**Phase 5 — Mobile Application**
Build a mobile client that connects to the existing backend, supporting chat, notifications, and summaries.

**Phase 6 — Testing & Wrap-Up**
End-to-end testing, final evaluation against the Phase 1 baseline, and documentation of results.
